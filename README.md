# All for Christ Zambia website

Official public website source for All for Christ Zambia.

## Local preview

Run any static file server in this directory, for example:

```sh
python3 -m http.server 8080
```

Then open `http://localhost:8080`.

## Publishing

The site is published through GitHub Pages after changes are reviewed and merged into `main`. Domain ownership remains with the ministry's registrar account.

## Content ownership

Do not commit passwords, registrar credentials, member records, unreleased audio files, contracts, or private ministry deliberations to this public repository.
